# 📋 Todo-List App (Node.js + Docker + Jenkins + Kubernetes)

Application de gestion de tâches avec CI/CD et déploiement Kubernetes.

## Fonctionnalités
- Création/modification/suppression de tâches
- Statut, priorité, date limite
- Pipeline CI/CD avec Jenkins
- Déploiement Kubernetes automatisé

## Lancer l'app
```bash
npm install
npm start
